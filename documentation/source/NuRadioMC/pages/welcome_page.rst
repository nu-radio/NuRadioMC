NuRadioMC is the **Monte-Carlo simulation** package of NuRadio.
To explore the different features available in NuRadioMC,
have a look at the :doc:`manuals </NuRadioMC/pages/manuals>` or the annotated examples in the ``examples`` subfolders. There is also a `webinar <https://www-zeuthen.desy.de/~anelles/NuRadioMC.mp4>`_ that provides a hands-on tutorial. Finally, the complete API documentation can be found under :doc:`Code Documentation </NuRadioMC/pages/code_documentation>`.
