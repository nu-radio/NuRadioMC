NuRadioReco is the **reconstruction** package of NuRadio.
To explore the different features available in NuRadioReco,
have a look at the :doc:`manuals </NuRadioReco/pages/howto_documentation>` or the annotated examples in the ``examples`` subfolders. There is also a `webinar <https://www-zeuthen.desy.de/~anelles/NuRadioReco.mp4>`_ that provides a hands-on tutorial. Finally, the complete API documentation can be found under :doc:`Code Documentation </NuRadioReco/pages/code_documentation>`.
